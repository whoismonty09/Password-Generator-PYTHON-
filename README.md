# Day 4 - Password Generator (Python)

## Description
This project is a simple Password Generator built using Python. It generates a strong and random password based on the length provided by the user. The password includes a combination of letters, numbers, and special characters to improve security.

## Features
- Generates random and secure passwords
- User can choose password length
- Uses letters, digits, and special characters
- Simple and easy to use
- Command-line based application

## Technologies Used
- Python
- random module
- string module
